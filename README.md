# 🐦 Seagull Bot — инструкция по запуску

## Что нужно сделать один раз

### 1. Создать бота в Telegram
1. Открой [@BotFather](https://t.me/BotFather)
2. Отправь `/newbot`
3. Придумай имя и username боту
4. Скопируй **BOT_TOKEN** (вида `123456:ABCdef...`)

### 2. Узнать свой Telegram ID
1. Открой [@userinfobot](https://t.me/userinfobot)
2. Нажми `/start`
3. Скопируй число из поля **Id** — это `ADMIN_CHAT_ID`

### 3. Добавить бота в канал
1. Зайди в настройки своего канала с чайками
2. Администраторы → Добавить администратора
3. Найди своего бота по username
4. Дай права на **публикацию сообщений**

### 4. Залить файлы на GitHub
1. Зайди на [github.com](https://github.com) → New repository
2. Назови, например, `seagull-bot`
3. Загрузи три файла: `bot.py`, `requirements.txt`, `nixpacks.toml`

### 5. Задеплоить на Railway
1. Зайди на [railway.app](https://railway.app) → New Project
2. Выбери **Deploy from GitHub repo** → выбери `seagull-bot`
3. Зайди в **Variables** и добавь три переменные:

| Переменная | Значение |
|---|---|
| `BOT_TOKEN` | токен от BotFather |
| `ADMIN_CHAT_ID` | твой числовой ID |
| `CHANNEL_ID` | `@username_канала` |

4. Нажми **Deploy** — готово! 🎉

---

## Как пользоваться

- Каждые **2 часа** бот проверяет Reddit и присылает тебе карточки с постами
- Нажимаешь **✅ В очередь** → пишешь описание следующим сообщением
- Бот автоматически публикует посты в канал в **12:00, 18:00 и 00:00 МСК**
- `/check` — проверить прямо сейчас
- `/status` — сколько постов в очереди
- `/publish` — опубликовать следующий вручную

## Добавить Threads

В файле `bot.py` найди раздел `FEEDS` и раскомментируй строку:
```python
"https://rsshub.app/threads/user/USERNAME",
```
Замени `USERNAME` на ник нужного аккаунта в Threads.

---

## Изменить время публикаций

В файле `bot.py` найди строку:
```python
PUBLISH_HOURS = "9,15,21"
```
Цифры — это часы по UTC. Для перевода в МСК прибавь 3.
Например, `"8,14,20"` = публикации в 11:00, 17:00 и 23:00 МСК.
